🚀 CosmoQuiz — CS Trivia Quiz App

Test your knowledge across the universe of Computer Science.

An interactive Flutter quiz app with 50 questions across 5 CS categories, random question selection, countdown timer, point system, animated starfield background, and a full results review screen.

✨ Features

🧠 50 Questions — 10 per category, 7 randomly picked per round
🎯 5 Categories — Flutter, Python, Algorithms, Systems, AI/ML
⏱️ 15-Second Timer — Color changes to red when time is running out
⚡ Point System — Different point values per question difficulty
🎨 Category Colors — Each category has its own color identity
🌟 Animated Starfield — Twinkling stars in the background
🪐 Orbit Animation — Rotating planet on landing screen
📊 Progress Dots — Visual progress bar across questions
🏆 Grade System — Cosmic Master, Star Explorer, Space Cadet, Keep Learning
📋 Review Screen — See correct/wrong answers after quiz
🔄 Play Again — Reshuffles 7 random questions each round

🎯 How It Works
1. Select a category (or All for mixed)
2. 7 random questions load from that category
3. Each question has 15 seconds — answer fast!
4. Green = correct, Red = wrong, timer shows remaining
5. Results screen shows score, grade and accuracy
6. Review tab shows all correct answers
7. Play Again reshuffles new random questions
